An automated security solution for financial institutions, designed to aggregate threat intelligence and enforce dynamic network defenses.

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Configure your `.env` file with `MONGO_URI`.
3. Run the dashboard: `python app.py`
